## 수정 사항 및 수정 해줘야 되는 것

로그인 기능 구현 완료.

1. 폰트 적용하기

2. 코드 최적화 및 오류 수정하기

3. 로그인 한 걸 바탕으로 데이터 저장 기능 및 페이지 마다 연결 기능 구현하기

4. 로그인 하면 로그아웃 버튼으로 바꾸게 수정 및 오른쪽 프로필 바 구현하기


## 문제점

로그인은 되는데 로그인을 하면 동주님이 열고있는 https://emojournal.djloghub.com/MainPage 사이트로 연결 돼서
코드를 수정해도 반영이 안되는 모습을 볼 수 있음. 그래서 일단 로그인을 하지 않은 상태의 페이지 자체에서 코드를 수정해야 웹페이지에 반영이 됨.
그렇게 해서 코드를 완성 시키고 나중에 통합 해야함.
